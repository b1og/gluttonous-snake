# [The Gluttonous Snake](https://github.com/yoonge/gluttonous-snake)

Gluttonous Snake Game built with TypeScript.

## Getting Started

```sh
git clone https://github.com/yoonge/gluttonous-snake.git
cd gluttonous-snake
pnpm i
pnpm start
```
